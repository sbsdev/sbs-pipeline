#+BEGIN_SRC sh

There is a bug in the cli in that it needs write access to
~/opt/daisy-pipeline/cli~. As a workaround do the following:

#+BEGIN_SRC sh
chmod o+w /opt/daisy-pipeline/cli
#+END_SRC sh
